# 交易底层逻辑与方法论

## 一、交易体系框架

### 1.1 三层交易系统
```
第一层：大势判断（能不能做）
  └── 大盘趋势 / 市场情绪 / 资金面

第二层：选股（做什么）
  └── 行业 / 个股 / 时机

第三层：交易执行（怎么做）
  └── 买点 / 卖点 / 仓位 / 风控
```

### 1.2 交易的底层逻辑
> 交易的本质是**概率游戏**。不追求每笔都赚，追求的是：**盈亏比 × 胜率 > 1**

核心公式：
- **期望值 = 胜率 × 平均盈利 - 败率 × 平均亏损**
- 哪怕胜率只有 40%，只要盈亏比达到 3:1，长期也能稳定盈利

## 二、四大主流方法论

### 2.1 趋势跟踪
**核心理念**：顺势而为，不预测只跟随

**判断工具**：
- 均线系统：MA5/MA10/MA20/MA60 多头排列 = 上升趋势
- 趋势线：连接低点画上升趋势线，跌破即趋势破坏
- ADX 指标：ADX > 25 表示趋势明确

**买卖规则**：
- 买入：均线多头排列 + 回踩均线不破
- 卖出：跌破关键均线或趋势线
- 加仓：突破前高确认趋势延续

**适用场景**：单边上涨/下跌行情

### 2.2 价值投资（A股版）
**核心理念**：买股票就是买公司，以合理价格买入优质资产

**A股特色选股标准**：
1. ROE 连续 3 年 > 15%
2. 净利润增长率连续 3 年 > 10%
3. PE 低于行业平均水平
4. 股息率 > 2%（有真金白银分红）
5. 市值 > 100 亿（避开小盘庄股）

**估值方法**：
- PE 估值法：适合盈利稳定的消费/医药股
- PB 估值法：适合银行/保险/券商等金融股
- PEG = PE / 净利润增长率，PEG < 1 表示低估

**安全边际**：在估算价值基础上打 7-8 折买入

### 2.3 波段操作
**核心理念**：在震荡市中高抛低吸，赚取波动差价

**关键要素**：
- 周期：通常 3-20 个交易日
- 振幅：选择日均振幅 > 3% 的活跃股
- 支撑阻力：在箱体底部买、顶部卖

**技术工具**：
- 布林带：下轨买入、上轨卖出
- RSI：RSI < 30 超卖买入，RSI > 70 超买卖出
- KDJ：K线上穿D线金叉买入

**适用场景**：横盘震荡行情
**不适用**：单边趋势行情（容易卖飞或深套）

### 2.4 题材炒作（A股特色）
**核心理念**：A股是政策市+资金市，题材驱动短期暴涨

**炒作阶段**：
1. **朦胧期**：政策刚出，先知先觉资金入场（最佳买点）
2. **发酵期**：媒体开始报道，散户关注（主升浪）
3. **高潮期**：全民讨论，龙头连续涨停（分批卖出）
4. **退潮期**：龙头炸板，跟风股大跌（清仓离场）

**选龙头标准**：
- 第一个涨停/最早封板
- 封单量最大
- 题材最正宗
- 流通市值适中（50-200亿最适合炒作）

## 三、量价关系核心法则

### 3.1 量价八法
| 量 | 价 | 信号 |
|----|----|------|
| 放量 | 上涨 | ✅ 健康上涨，继续看多 |
| 缩量 | 上涨 | ⚠️ 上涨乏力，警惕回调 |
| 放量 | 下跌 | ❌ 恐慌抛售，远离 |
| 缩量 | 下跌 | 🔄 下跌减速，可能见底 |
| 放量 | 横盘 | ⚠️ 多空分歧，变盘前兆 |
| 缩量 | 横盘 | 🔄 交投清淡，等待方向 |

### 3.2 关键量价信号
- **底部放量长阳**：反转信号，重点跟进
- **高位放量长阴**：见顶信号，立即减仓
- **缩量涨停**：筹码锁定好，次日大概率高开
- **放量滞涨**：主力出货，要小心

## 四、筹码分布分析

### 4.1 核心概念
- **筹码密集区**：大量持仓成本集中的价格区间
- **筹码峰**：筹码分布图中的峰值区域
- **套牢盘**：持仓成本高于当前价的筹码
- **获利盘**：持仓成本低于当前价的筹码

### 4.2 实战应用
- 股价从底部突破筹码密集区 → 主力完成建仓，上涨可期
- 股价在高位、下方筹码峰消失 → 主力已出货，要跑
- 90% 筹码集中度 < 10% → 筹码高度集中，随时可能拉升

## 五、交易系统设计框架

### 5.1 设计你的交易系统
```
1. 确定交易风格（趋势/波段/短线/长线）
2. 确定入场条件（什么情况买？）
3. 确定出场条件（什么情况卖？）
4. 确定仓位规则（买多少？）
5. 确定风控规则（亏多少必须走？）
6. 历史回测验证
7. 实盘小资金验证（至少 20 笔）
8. 正式投入使用
```

### 5.2 交易系统评估标准
- 年化收益率 > 15%
- 最大回撤 < 20%
- 胜率 > 40%
- 盈亏比 > 2:1
- 夏普比率 > 1

**如果一套系统能在历史回测中满足以上条件，才值得投入实盘。**
