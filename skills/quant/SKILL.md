# A股量化交易实战

## 前置依赖

本 skill 依赖以下已安装的 skills：
- **tongdaxin-mcp**：提供行情/选股/研报数据
- **neodata-financial-search**：提供财务/资金流向数据

## 一、数据获取流程

### 1.1 通过通达信获取实时行情
使用 `tdx-finance_qclaw` MCP 服务获取：
- 实时行情（价格、涨跌幅、成交量）
- K 线数据（日/周/月/分钟）
- 板块行情数据
- 龙虎榜数据

### 1.2 通过 NeoData 获取财务数据
使用自然语言查询：
- 财务报表（利润表/资产负债表/现金流量表）
- 财务指标（PE/PB/ROE/毛利率等）
- 资金流向（主力净流入/北向资金）
- 研报评级

## 二、多因子选股模型

### 2.1 因子体系
```
【价值因子】
- PE（市盈率）< 行业平均 70%
- PB（市净率）< 行业平均
- 股息率 > 2%

【成长因子】
- 营收增长率（近 3 年）> 15%
- 净利润增长率（近 3 年）> 20%
- ROE > 15%

【质量因子】
- 毛利率 > 30%
- 资产负债率 < 60%
- 经营现金流/净利润 > 0.8

【动量因子】
- 近 1 个月涨幅 > 0
- 近 3 个月涨幅 > 5%
- 近 6 个月涨幅排名全市场前 30%

【技术因子】
- MA5 > MA20（短期多头）
- 成交量放大（日均量 > 20日均量 × 1.5）
- RSI(14) 在 40-60 区间（不超买不超卖）
```

### 2.2 选股流程
```
Step 1: 用通达信筛选全市场（排除ST、亏损股）
Step 2: 用 NeoData 查询财务指标，过滤
Step 3: 用多因子打分排序
Step 4: 取前 20-30 只进入观察池
Step 5: 人工复核（排除庄股、题材炒作过度的）
```

## 三、经典量化策略

### 3.1 双均线策略
```
买入条件：MA5 上穿 MA20 + 成交量放大
卖出条件：MA5 下穿 MA20 或 跌破 MA60
仓位：首次买入 30%，确认趋势加仓到 60%
止损：跌破买入日最低价 -3%
```

### 3.2 网格交易策略
```
适用场景：震荡市，挑选波动大的 ETF/蓝筹股
网格间距：3-5%（根据个股历史波动率设定）
网格层数：5-8 层
底仓：总计划资金的 50% 一次性买入
每层金额：总计划的 10%
```

### 3.3 动量突破策略
```
选股条件：
  - 近 20 日涨幅排名全市场前 10%
  - 当日突破 20 日新高
  - 成交量 > 20日均量 × 2

买入：突破确认当天尾盘买入
止损：突破当日最低价
止盈：持有 5-10 个交易日，不设固定止盈
仓位：每只等额分配，总仓不超过 50%
```

### 3.4 龙虎榜跟踪策略
```
关注信号：
  - 机构席位净买入 > 5000 万
  - 知名游资席位买入（如：中信上海分公司）
  - 多家机构同时买入
  - 买一至买五合计 > 卖一至卖五合计 × 2

操作：
  - 次日如果高开 ≤ 3%，可以跟进
  - 如果高开 > 5%，观望
  - 买入后持有 1-3 天
  - 止损 -5%
```

## 四、风控量化指标

### 4.1 核心风控参数
| 指标 | 保守 | 均衡 | 激进 |
|------|------|------|------|
| 单票最大仓位 | 10% | 20% | 30% |
| 单行业最大仓位 | 20% | 30% | 50% |
| 总仓位上限 | 60% | 80% | 100% |
| 单日最大亏损 | 2% | 3% | 5% |
| 单票止损线 | -5% | -8% | -12% |
| 持仓数量 | 3-5 | 5-8 | 8-15 |

### 4.2 回撤控制铁律
- 账户回撤达 10%：仓位减半
- 账户回撤达 15%：停止开新仓，只平仓
- 账户回撤达 20%：清仓休息至少 1 个月

## 五、北向资金跟踪

北向资金（沪股通+深股通）是 A 股重要的情绪指标：

- **连续 5 日净流入 > 100 亿**：大盘中期看多
- **连续 3 日净流出 > 50 亿**：警惕调整风险
- **单日净流入 > 200 亿**：短期情绪高点，适当减仓
- **单日净流出 > 100 亿**：恐慌信号，观察是否持续

重点关注北向资金净买入最多的前 10 只个股，这些往往是外资认可的优质标的。

## 六、回测框架

### 6.1 回测基本要素
```
起始资金：根据实际设定
回测周期：至少 3 年（覆盖牛熊市）
交易成本：印花税 0.1% + 佣金 0.03% + 滑点 0.1%
基准对比：沪深 300 指数
```

### 6.2 回测评估标准
```
年化收益率 > 基准 + 5%
最大回撤 < 基准最大回撤
夏普比率 > 1.2
胜率 > 45%
盈亏比 > 2.0
月胜率 > 55%
```

### 6.3 过拟合防范
- 策略参数不超过 3 个
- 样本外测试至少 1 年数据
- 不同市场环境下分别测试
- 实盘小资金跑 1-3 个月验证
