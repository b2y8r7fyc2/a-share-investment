# A股ETF大波段拐点识别体系

> **核心理念**：月线定方向，周线定节奏，日线定点位。
> **交易本质**：三层共振时是最大波段机会，三层背离时是最大风险。

## 决策框架（三层嵌套）

```
第一层：宏观层 → 决定总仓位上限
  └── 流动性 + 信用周期 + 政策（持续数月到数年）

第二层：中观层 → 决定板块配置方向
  └── 产业景气度 + 盈利周期（持续数周到数月）

第三层：微观层 → 决定具体进出时点
  └── 资金流动 + 情绪 + 技术形态（持续数天到数周）
```

## 投资者画像（默认参数）

| 参数 | 设定 |
|------|------|
| 风格 | 中线波段（持仓≥2周），只做老登股/蓝筹ETF，不碰游资题材 |
| 单日最大回撤 | 5% |
| 年度收益目标 | 20% |
| 单板块上限 | S级25%、A级20%、B级10%、C级5% |
| 不碰品种 | 军工、传媒、游戏、次新 |

## 数据获取

**默认使用** neodata-financial-search（自然语言查询行情/财务/资金流向）。若 tongdaxin-mcp 已授权则作为 K 线/选股补充源。禁止用训练数据替代实时数据。

## 输出规范

1. **不幻想**：所有判断必须标注数据来源和获取时间
2. **不讨好**：该清仓就说清仓，不因用户持仓而偏袒
3. **不摇摆**：基于评分卡给出结论，结论应与评分一致
4. **可校验**：每个结论附带可验证的量化指标
