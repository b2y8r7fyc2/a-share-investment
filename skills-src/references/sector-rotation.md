# 板块轮动规律

## 一、宏观周期轮动

| 阶段 | 宏观环境 | 领涨板块 | 操作重点 |
|------|----------|----------|----------|
| 流动性宽松初期 | 降息/降准落地 | **科技成长**（储能、半导体、AI算力） | 重仓成长 |
| 经济复苏确认 | PMI连续回升 | **周期股补涨**（有色、化工、机械） | 逐步切换 |
| 过热紧缩 | 加息/收紧 | **防御避险**（绿电、公用事业、红利） | 减仓防守 |

---

## 二、板块内部轮动节奏

### 科技主线内部扩散
```
算力硬件（存储/光模块/通信）  ← 第一波领涨
    ↓
半导体设备/材料               ← 第二波扩散
    ↓
AI应用（软件/智能体）         ← 第三波扩散
    ↓
消费电子（端侧AI）            ← 第四波扩散
```

### 电力链内部轮动
```
绿色电力（防御底仓）         ← 先走强
    ↓
电网设备（特高压）           ← 接力
    ↓
电池储能（弹性最大）         ← 接力
    ↓
新能源车（终端需求）         ← 最后接力
```

---

## 三、轮动信号识别

| 信号 | 含义 | 操作 |
|------|------|------|
| 领涨板块滞涨 + 低位板块放量 | 轮动即将发生 | 减高位配低位 |
| 领涨板块成交量占比>15% | 情绪过热，轮动在即 | 减持领涨板块 |
| 低位板块持续缩量 | 轮动条件不具备 | 不急于切换 |
| 高位板块龙头破位 | 主线切换确认 | 清仓高位板块 |

## 五、情绪周期定位

市场永远沿以下五阶段循环：
```
恐慌 → 怀疑 → 确认 → 贪婪 → 疯狂
  │                                    │
  └──── 最佳买点             最佳卖点 ──┘
```

| 阶段 | 特征 | 操作 |
|------|------|------|
| **恐慌** | 全市场下跌，情绪崩溃 | 不割肉，开始观察 |
| **怀疑** | 反弹了但不敢信 | 调结构，试探建仓 |
| **确认** | 信号明确，资金流入 | 加仓至目标仓位 |
| **贪婪** | 人人看好，媒体推荐 | 分批止盈 |
| **疯狂** | 涨停潮，热搜密集 | 清仓离场 |

### 当前定位（2026年6月下旬）
处于**确认后期 → 即将进入贪婪期**。此时应持有但设好止盈，不新增重仓。
