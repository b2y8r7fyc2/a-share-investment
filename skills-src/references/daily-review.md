# 每日复盘模板

## 盘前准备（9:00前）

- [ ] 隔夜外盘走势（美股三大指数、A50期货）
- [ ] 重大消息面（政策、公告、宏观数据）
- [ ] 今日操作计划（哪些持有、哪些关注、哪些准备操作）

---

## 盘中纪律

- [ ] 严格执行"延5切10"止损止盈，收盘前5分钟确认
- [ ] 记录每笔交易（时间、价格、理由、情绪状态）
- [ ] 不因情绪改变计划
- [ ] 宏观事件前只设条件单，不主动操作
- [ ] 当日净值波动>2% → 强制冷静1小时

---

## 盘后复盘（15:30后）

### 1. 纪律执行检查
- [ ] 是否违反8条铁律中的任何一条？
- [ ] 单板块亏损是否达-5%？是否已执行清仓？
- [ ] 总组合回撤是否达-3%？是否已复盘写书面记录？
- [ ] 是否有"手痒式交易"（无信号开仓、追涨杀跌）？
- [ ] 月度操作是否≤8笔？单笔金额是否≥总仓位1%？

### 2. 劣质仓位检查
- [ ] 是否有D/F级板块仍在组合中？
- [ ] 电网、储能等被连续点名的板块是否已清仓？
- [ ] 防御核心仓位是否被错误削减？

### 3. 拐点信号扫描
对每个持仓/关注板块，更新六个维度状态：

| 板块 | 技术面 | 情绪面 | 消息面 | 政策面 | 资金面 | 筹码面 | 评分变化 |
|------|:---:|:---:|:---:|:---:|:---:|:---:|:---:|
| | | | | | | | |

### 4. 强弱排序
- 按底部评分从高到低排列 → 确定加仓优先级
- 按顶部风险评分从高到低排列 → 确定减仓优先级

### 5. 明日操作计划
| 标的 | 方向 | 仓位变动 | 金额 | 依据（引用纪律编号） |
|------|:---:|:---:|------|------|
| | | | | |

---

## 每周复盘（周五盘后）

### 周度回顾
- 本周收益/亏损
- 正确决策（应固化）
- 错误决策（应记录原因和教训）
- 纪律执行率（正确执行次数/总操作次数）

### 体系迭代
- 是否有新规律需要写入体系？
- 是否有旧规则需要修正？
- 评分卡权重是否需要调整？

---

## 看板刷新流程

每次要求"刷新策略看板"时，执行以下步骤：

```
1. 数据拉取：用 neodata-financial-search 获取所有关注板块最新数据
2. 跑评分卡：对每个板块跑底部评分卡 + 持仓板块跑顶部风险评分卡
3. 定位波浪：用波浪理论定位每个板块所处阶段
4. 检查纪律：逐条检查持仓是否触发任何纪律条件
5. 排序输出：板块打分排序 + 左右侧分类 + 仓位建议
6. 预测输出：未来两周走势预判 + 板块涨幅锐度排序
7. 操作输出：明确明日加减仓操作及纪律依据
```
