# 量化评分卡

## 一、波段底部评分卡（满分100）

| 信号组 | 权重 | 计分规则 |
|:---|:---:|:---|
| **估值压缩** | 10分 | PE分位<10%=10分，10-15%=6分，15-20%=3分 |
| **月线底背离** | 25分 | 双重背离25分，单一背离15分，未确认8分 |
| **月线倍量站上20月线** | 20分 | 放量突破20月线20分，缩量站上10分，未站上0分 |
| **周线多头排列** | 15分 | 5/10/20周线多头15分，金叉10分，粘合5分 |
| **领先股出现** | 15分 | ≥2只龙头创半年新高15分，1只8分，无0分 |
| **宏观景气** | 10分 | 行业核心指标连续2月转好10分，转好5分，恶化0分 |
| **资金流向** | 5分 | 连续3周净流入5分，1-2周3分，净流出0分 |

### 底部评分决策矩阵

| 总分 | 信号 | 仓位 |
|:---:|------|:---:|
| ≥85 | 🔥 高概率大底 | 重仓出击，单板块可至20-25% |
| 75-84 | ✅ 底部区域 | 分批建仓，单板块10-15% |
| 60-74 | ⏳ 底部萌芽 | 观察仓位3-5%，等信号升级 |
| <60 | 🚫 不构成底部 | 不持有，反弹即走 |

## 二、波段顶部风险评分卡（满分100）

| 信号组 | 权重 | 计分规则 |
|:---|:---:|:---|
| **估值极端** | 15分 | PE分位>90%=15分，85-90%=12分，80-85%=8分 |
| **月线顶背离** | 30分 | 价格新高+MACD红柱萎缩30分，背离未确认15分 |
| **跌破10月线** | 20分 | 大波段结束硬止损信号 |
| **周线死叉** | 15分 | 5/10周死叉+跌破30周线15分，仅死叉10分 |
| **情绪过热** | 10分 | 成交占比>15%或热搜密集10分，偏高5分 |
| **龙头破位** | 10分 | 最强股月跌>20% =10分，>10%=5分 |

### 顶部风险决策矩阵

| 总分 | 信号 | 操作 |
|:---:|------|------|
| ≥80 | 🔴 高概率大顶 | 清仓，不留任何仓位 |
| 60-79 | ⚠️ 风险区域 | 减仓至5%以下，设硬止损 |
| 40-59 | ⚠️ 警惕信号 | 减仓50%，阶梯止盈剩余仓位 |
| <40 | ✅ 正常 | 按趋势持有 |

## 三、板块强弱判断公式

> 板块强度 = 技术面(40%) + 资金面(25%) + 政策面(20%) + 情绪面(15%)

| 维度 | 权重 | 核心指标 |
|------|:---:|------|
| 技术面 | 40% | 周线MACD方向 + 均线排列 + 底背离信号 |
| 资金面 | 25% | 主力连续净流入天数 + 成交占比变化 |
| 政策面 | 20% | 近期政策催化密度 + 政策级别 |
| 情绪面 | 15% | 板块成交额/全市场成交额占比 + RSI位置 |

---

## 四、使用流程

```
1. 获取各板块最新行情数据（neodata-financial-search）
2. 对每个板块跑底部评分卡 → 得到底部得分
3. 对持仓板块跑顶部风险评分卡 → 得到风险得分
4. 按底部得分排序 → 确定加仓优先级
5. 按顶部风险得分排序 → 确定减仓/清仓优先级
6. 输出：板块打分排序表 + 左右侧分类 + 仓位建议
```
