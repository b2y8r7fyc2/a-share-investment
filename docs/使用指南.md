# A股投资工具箱 - 主入口

## 使用说明

向 OpenClaw 说出你的需求，AI 会自动调用对应 skill。

### 行情查询
- "查询宁德时代最新行情" → tongdaxin-mcp
- "今天A股涨跌停板有哪些" → tongdaxin-mcp
- "半导体板块今天资金流向" → neodata-financial-search

### 选股分析
- "用多因子模型帮我筛选优质A股" → quant-trading + neodata
- "最近北向资金在买什么" → neodata-financial-search
- "帮我看看茅台的基本面" → tongdaxin-mcp + neodata

### 策略参考
- "现在适合用双均线策略吗" → trading-methodology
- "帮我设计一个网格交易计划" → trading-methodology
- "龙虎榜有什么值得关注的" → quant-trading

### 交易纪律
- "我今天亏了5%，该怎么办" → trading-discipline
- "帮我做今天的复盘" → trading-discipline
- "怎么控制自己不追涨杀跌" → trading-discipline

### 基金投资
- "推荐几个适合定投的ETF" → fund-swing-investment
- "医疗ETF现在能买吗" → fund-swing-investment
- "帮我评估一下张坤的基金" → fund-swing-investment

### 综合咨询
- "当前市场环境下如何配置资产" → 多 skill 协同
- "帮我做一份完整的投资计划" → 所有 skill 协同
